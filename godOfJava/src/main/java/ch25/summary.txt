쓰레드
[1] 개념
    * 하나의 프로세스에 여러개의 쓰레드 존재
    * main() 실행시, 하나의 쓰레드 수행
    * main() 에서 쓰레드 생성해줌

쓰레드가 만들어진 이유
* 효율적으로 메모리 공간을 사용하기 위해서

[2] 쓰레드를 생성하는 방법
    * 방법 1) Runnable 인터페이스 사용
        - implements Runnable -> run() 메소드 구현
        - Runnable을 구현한 클래스 객체 생성
        - 쓰레드 클래스를 생성하고 runnable을 생성자의 매개변수에 넣는다.
        - 생성한 쓰레드 객체에 start()를 호출한다.
    * 방법 2) Thread 클래스 사용
        - 쓰래드를 생성한다.
        - 쓰래드의 start메소드를 호출한다.

    * 공통점
        - java.lang에 존재
        - void run() 메소드만 존재
    * 차이점
        - 쓰레드를 실행하는 방식
